$ \mathcal f (x; a_1; a_2) \Rightarrow y = a_1x^{{'{'}}a_2{{'}'}} $

En este caso la función no es lineal por lo cual debemos linealizarla:
$ ln \ y = ln \ a_1 + ln \ x^{{'{'}}a_2{{'}'}} \Rightarrow ln \ y = ln \ a_1 + a_2 \ ln \ x $

Comparando la expresión anterior con la fórmula empírica de la recta nos queda:
$ \begin{{'{'}}align{{'}'}} y = a_1 + a_2x && ln \ y = ln \ a_1 + a_2 \ln \ x \\ y \Rightarrow ln \ y && a_2 = a_2\\ a_1 \Rightarrow ln \ a_1 && x \Rightarrow ln \ x \end{{'{'}}align{{'}'}} $

Tomando en cuenta el Análisis Inicial (Ver análisis inicial). Resulta:
$ \begin{{'{'}}bmatrix{{'}'}} n & \sum_{{'{'}}i=0{{'}'}}^n ln \ x_i \\ \sum_{{'{'}}i=0{{'}'}}^n ln \ x_i & \sum_{{'{'}}i=0{{'}'}}^n ln \ x_i^2 \\ \end{{'{'}}bmatrix{{'}'}} \cdot \begin{{'{'}}bmatrix{{'}'}} ln \ a_1\\ a_2\\ \end{{'{'}}bmatrix{{'}'}} = \begin{{'{'}}bmatrix{{'}'}} \sum_{{'{'}}i=0{{'}'}}^n ln \ y_i\\ \sum_{{'{'}}i=0{{'}'}}^n ln \ y_i \cdot ln \ x_i\\ \end{{'{'}}bmatrix{{'}'}} $

Tomamos como ejemplo la tabla de valores ingresada.
Ajustamos a una exponencial: $ y = a_1 x^{{'{'}}a_2{{'}'}} $

$ \begin{{'{'}}align{{'}'}} n = {{n}} && \sum_{{'{'}}i=1{{'}'}}^n ln \ x_i = {{csx}} && \sum_{{'{'}}i=1{{'}'}}^n ln \ x_i^2 = {{csx2}} && \sum_{{'{'}}i=1{{'}'}}^n ln \ y_i = {{csy}} && \sum_{{'{'}}i=1{{'}'}}^n ln \ y_i \cdot ln \ x_i = {{csyx}} \end{{'{'}}align{{'}'}} \\ \begin{{'{'}}bmatrix{{'}'}} \ n & \sum_{{'{'}}i=1{{'}'}}^n ln \ x_i \\ \sum_{{'{'}}i=1{{'}'}}^n ln \ x_i & \sum_{{'{'}}i=1{{'}'}}^n ln \ x_i^2 \end{{'{'}}bmatrix{{'}'}} \cdot \begin{{'{'}}bmatrix{{'}'}} ln \ a_1\\ a_2 \end{{'{'}}bmatrix{{'}'}} = \begin{{'{'}}bmatrix{{'}'}} \sum_{{'{'}}i=1{{'}'}}^n ln \ y_i\\ \sum_{{'{'}}i=1{{'}'}}^n ln \ y_i \cdot ln \ x_i \end{{'{'}}bmatrix{{'}'}} \Rightarrow \begin{{'{'}}bmatrix{{'}'}} {{n}} & {{csx}} \\ {{csx}} & {{csx2}} \end{{'{'}}bmatrix{{'}'}} \cdot \begin{{'{'}}bmatrix{{'}'}} ln \ a_1\\ a_2 \end{{'{'}}bmatrix{{'}'}} = \begin{{'{'}}bmatrix{{'}'}} {{csy}}\\ {{csyx}} \end{{'{'}}bmatrix{{'}'}} $
